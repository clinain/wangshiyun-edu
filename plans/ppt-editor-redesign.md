# PPT编辑器 WPS 风格重新设计计划

## 当前状态分析

当前 PPT 编辑器是一个简单的表单式编辑器：
- 左侧：页面列表（纯文本列表）
- 右侧：表单输入（标题、内容、备注等文本框）
- 顶部：简单的操作栏

## 目标设计

参考 WPS Office PPT 编辑器的布局，重新设计为：

### 布局结构

```
+----------------------------------------------------------+
|  顶部工具栏：文件名 | 撤销/重做 | 模板 | 预览 | 保存      |
+--------+------------------------------------+-----------+
|        |                                    |           |
| 幻灯片 |       主编辑区域                    |  属性面板  |
| 缩略图  |    渲染当前幻灯片预览               | 页面设置   |
|        |    显示标题+内容+布局效果           | 字体/颜色  |
|  [+]   |                                    | 背景设置   |
|        |                                    |           |
+--------+------------------------------------+-----------+
|  状态栏：当前页码 / 总页数 | 模板风格                         |
+----------------------------------------------------------+
```

### 详细组件设计

#### 1. 顶部工具栏 `PPTToolbar`
- 左侧：PPT 标题（可编辑输入框）
- 中间：撤销/重做按钮、模板风格选择下拉
- 右侧：预览按钮、保存按钮

#### 2. 左侧幻灯片缩略图面板 `SlideThumbnails`
- 显示所有幻灯片的缩略图预览
- 每个缩略图渲染实际的幻灯片效果（标题+内容）
- 点击切换当前编辑页面
- 支持拖拽排序
- 右键菜单：复制、删除、上移、下移
- 底部"+"按钮添加新幻灯片
- 页面类型图标标识（封面、内容、目录、结束）

#### 3. 主编辑区域 `SlideEditor`
- 显示当前幻灯片的渲染预览效果
- 类似 WPS 的所见即所得视图
- 显示幻灯片的实际布局效果：
  - 封面页：大标题 + 副标题居中
  - 内容页：标题 + 要点列表
  - 目录页：章节列表
  - 结束页：感谢文字居中
- 点击标题/内容区域可直接编辑（contentEditable 或点击弹出编辑框）
- 背景色根据模板风格变化

#### 4. 右侧属性面板 `PropertyPanel`
- 页面类型选择
- 字体大小调整
- 背景颜色选择
- 演讲备注编辑

#### 5. 底部状态栏
- 当前页码 / 总页数
- 当前模板风格名称

### 页面类型与布局映射

| 页面类型 | 缩略图效果 | 主编辑区效果 |
|---------|-----------|-------------|
| 封面页 cover | 大标题居中 + 副标题 | 渐变背景 + 大号标题 + 小号副标题 |
| 目录页 toc | 章节列表 | 编号列表展示 |
| 内容页 content | 标题 + 要点 | 标题栏 + bullet points |
| 结束页 end | 感谢文字 | 大号居中文字 |

### 实现步骤

1. **创建 `PPTToolbar` 组件** - 顶部工具栏
2. **创建 `SlideThumbnails` 组件** - 左侧缩略图面板
3. **创建 `SlideEditor` 组件** - 主编辑区域（渲染幻灯片预览）
4. **创建 `PropertyPanel` 组件** - 右侧属性面板
5. **重构 `PPT.tsx` 中的编辑器部分** - 组合以上组件
6. **添加幻灯片渲染逻辑** - 根据页面类型和内容渲染预览
7. **保持 API 兼容** - 确保保存/加载逻辑不变

### 技术方案

- 使用 React 函数组件
- 使用 Tailwind CSS 样式
- 幻灯片预览使用 CSS 渲染（非 canvas）
- 缩略图使用缩小比例的相同渲染
- 保持现有的 `CustomPage` 数据结构
- 保持现有的 `pptAPI` 保存逻辑

### 不变的部分

- 从教案生成功能（顶部区域）
- 我的PPT列表页面
- PPT 详情页预览
- 后端 API 和数据模型
